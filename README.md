# Задание 1

## Задача 1

### Текст задачи

На основе данного списка строк получить новый, где к каждой строке в конце

дописан заданный символ.

### Алгоритм решения

Пользователь вводит число n - число элементов списка. Далее запрашиваем у пользователя n строк, сохраняем каждую в список. После, просим ввести символ. Данный символ будет добавлен в конец каждой строки списка (принимаются только одиночные СИМВОЛЫ), получившийся список выводим пользователю 

### Тестирование

Скриншоты результата работы программы



## Задача 2

### Текст задачи

Список содержит строки. Найти самую короткую строку

### Алгоритм решения

Пользователь вводит число n - число элементов списка. Далее запрашиваем у пользователя n строк, сохраняем каждую в список. После этого выводим длину самой короткой строки. Минимальная длина находится через сравнение длин строк

### Тестирование

Скриншоты результата работы программы
